The words you searched before are important elements of a Graphical User Interface (GUI). Read the definition of a GUI:

A GUI or a Graphical User Interface is a visual way for users to interact with electronic devices or computers through graphical icons, buttons, and other visual indicators. It allows users to interact with software applications using graphical elements such as windows, buttons, menus, and icons rather than text-based commands.

The GUI elements are used to give commands and guide you with the interaction in software applications. Look at some command sentences that you may find using a GUI:

“Enter your username and password, then click 'Login' to access your account” “Click the 'Save' button to save your changes” "Scroll down to see more content”

As you saw in the examples above, all the sentences start with a verb which express a command. This content in English is called Imperative verbs. Look at the chart below and learn about what imperative verbs are:

The imperative verb is an action a speaker or writer wants someone else to do. For this reason, they're occasionally called “command words” or “bossy verbs,” even though that may not be the speaker's or writer's intent. Look at different usages:

ODBC

ODBC, ou Open Database Connectivity, é um conjunto de classes-modelo para conexão disponível em qualquer linguagem de programação, diferindo-se da JDBC, que é exclusiva para Java.

JDBC

JDBC, ou Java EE Database Connectivity, é uma API (application programming interfaces) que possibilita a interação do código Java com um banco de dados, na qual as instruções SQL são executadas por meio de um driver, composto por um grupo de classes e interfaces desenvolvido em Java. Esse processo possibilita a execução de instruções SQLs que se conectarão com a base de dados, permitindo a manipulação dos valores.
